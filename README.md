# Realty Financial Inc Website

A simple, modern, professional marketing site for **Realty Financial Inc**, a real estate and mortgage brokerage led by **George Espinoza**.

Built with:

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open `http://localhost:3000` in your browser.

## Structure

- `app/` – App Router pages and layout
- `components/` – Reusable UI components
- `data/` – Mock data for closed deals
- `public/brand-bg.png` – Site-wide decorative background (used in root layout)

To adjust how strong the background graphic appears, edit the opacity layers in `app/layout.tsx`.

This starter is intentionally focused, with room to grow into a fuller experience later.
